# Project Lighthouse

Project Lighthouse es un sistema de desarrollo profesional diseñado para formar un líder en Arquitectura Empresarial y Gobierno de IA durante un horizonte de 10 años.

El proyecto se basa en la producción continua de entregables profesionales, la reflexión estructurada y el aprendizaje mediante casos reales.

## Objetivo

Convertir a Federico en un profesional capaz de liderar decisiones tecnológicas de alto impacto.

## Principios

- Deliverables over Study
- Finished > Perfect
- One Sprint, One Goal
- Evidence over Opinion
- System over Motivation
